# Data Insights (Simple)

Analyze any data and get 5 key questions answered in plain English.

## What it does

→ Hit targets?
→ Trending up or down?
→ Any unusual changes?
→ Best/worst segments?
→ Key numbers (avg, total)

Clean Excel output ready to share.

## How to use

1. Upload this skill to a Claude Project
2. Upload your CSV or Excel file
3. Say "analyze this data"

That's it.

## What it handles

- Sales data (revenue, units, customers)
- Health/fitness tracking
- Marketing metrics
- Budget vs actual
- Any tabular data with numbers

Works best when you have:
- Numeric columns to analyze
- Optional: Target/Goal columns
- Optional: Date/time columns for trends
- Optional: Categories (stores, products, regions)

## Output

3-tab Excel file:
- **Summary** - 3-5 key bullets + one action
- **Full Analysis** - All 7 analyses in plain English
- **Raw Data** - Your original data

Built for anyone who needs quick insights without being a data scientist.
